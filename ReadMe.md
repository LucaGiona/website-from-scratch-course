## 🌟 **Features**
- 🔥 **Modern CSS Techniques:** Custom Properties, Nesting, CSS Grid Layouts, and Utility Classes.  
- 📱 **Responsive Design:** Optimized for Desktop, Tablet, and Mobile Devices.  
- 🧩 **Advanced Layouts:** Bento-style layouts and complex grid systems.  
- 🎨 **Animations and Transitions:** Add interactive polish with custom view transitions.  
- 🚀 **Netlify Deployment:** Automated deployment using GitHub Actions and Netlify.  
- 🎯 **Professional Styling:** Showcase your CSS expertise with clean and maintainable code.  

---

## 🛠️ **Tech Stack**
- **HTML** – Website structure  
- **CSS (with CSS Layers)** – Design and layout  
- **JavaScript** – Interactive functions  
- **GitHub Actions** – CI/CD pipeline for automated deployment  
- **Netlify** – Hosting and deployment  

---

## 🚀 **Installation & Setup**
1. **Clone the Repository**  
```bash
git clone https://github.com/LucaGiona/website-from-scratch-course
cd course-description
```

2. **Install Dependencies**  
```bash
npm install
```

3. **Start Development Server**  
```bash
npm start
```
**Or with Visual Studio Code Live Server:**
- Open the project in VSCode.
- Click on **„Go Live“** in the status bar.

## 📚 **Usage**
- Navigate through the main navigation to explore the sections **"Discover"**, **"Mushroom Guide"**, and **"FAQ"**.  
- **Custom Properties** are used to manage color schemes and font sizes.  
- **CSS Grid** and **Flexbox** are utilized for advanced layout designs.
