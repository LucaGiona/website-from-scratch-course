🎨 Course Description – Write CSS like a Pro!
Course by Kevin Powell on Frontendmasters

Build a responsive website from scratch using the latest CSS features like Custom Properties, Nesting, CSS Grid Layouts, and Utility Classes. Add polish with Animations, Custom View Transitions, Advanced Bento-style Layouts, and Filtering. Deploy your site with a GitHub Actions Workflow to Netlify and show off your professional CSS skills!

🎓 About the Course
This project is part of a course from Frontendmasters, which I highly recommend for anyone looking to level up their frontend skills. Frontendmasters offers high-quality courses by industry experts and is perfect for both beginners and advanced developers.

🌟 Features
🔥 Modern CSS Techniques: Custom Properties, Nesting, CSS Grid Layouts, and Utility Classes.
📱 Responsive Design: Optimized for Desktop, Tablet, and Mobile Devices.
🧩 Advanced Layouts: Bento-style layouts and complex grid systems.
🎨 Animations and Transitions: Add interactive polish with custom view transitions.
🚀 Netlify Deployment: Automated deployment using GitHub Actions and Netlify.
🎯 Professional Styling: Showcase your CSS expertise with clean and maintainable code.
🛠️ Tech Stack
HTML – Website structure
CSS (with CSS Layers) – Design and layout
JavaScript – Interactive functions
GitHub Actions – CI/CD pipeline for automated deployment
Netlify – Hosting and deployment
